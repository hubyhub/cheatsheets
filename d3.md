## D3 Cheatsheet 

var dataset = [5, 10, 15, 17, 25];
```javascript
d3.select('body').selectAll('div')
.data(dataset)
.enter()
.append('div')
.attr('class','bar')
.style('height',function(d){
 return d*5+'px';
});
```

#### Explanation:
Der Code oben erzeugt einen simplen "column"-Chart basierend auf dem *dataset*.

**selectAll('div'):** liefert ein Array von allen DIVs die gefunden werden können. In diesem Fall ist das Array leer. 

**data(dataSet) oder data( dataSet, keyFunction )**:  definiert zum einen das dataSet-Array, zum anderen die keyFunction (optional), welche die Keys für die einzelnen Elemente des dataSetArrays bestimmt. 
WIrd **keine** keyFunction angegeben, so sind die Keys automatisch die Indices des dataSet-Arrays.
Also für [5, 10, 15, 17, 25] wären es [0, 1, 2, 3, 4].
Danach werden die dataSet-Keys mit den Keys der Elemente der **selectAll**-Funktion verglichen. Alle data-keys die *unterschiedlich* sind, werden dann Teil der  **enter**-Selection und bei der **enter()**-Funktion aufgerufen. 
Siehe "[knowledgestockpile](http://knowledgestockpile.blogspot.co.at/2012/01/understanding-selectall-data-enter.html)"

**enter().append('div'):** Aus allen Elementen in der Enter-Selection, werden mit Hilfe der enter().append() Funktion neue Nodes erzeugt. In diesem Fall werden neue DIVs erzeugt.

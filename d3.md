## D3 Cheatsheet 

var dataset = [5, 10, 15, 17, 25];
```javascript
d3.select('body').selectAll('div')
.data(dataset)
.enter()
.append('div')
.attr('class','bar')
.style('height',function(d){
 return d*5+'px';
});
```

#### Explanation:
Der Code oben erzeugt einen simplen "column"-Chart basierend auf dem *dataset*.

**selectAll('div'):** liefert ein Array von allen DIVs die gefunden werden können. In diesem Fall ist das Array leer. 

**data(dSet) oder data( dSet, keyFunction )**:  definiert zum einen das dataArray, zum einen die keyFunction, welche die Keys für die einzelnen Elemente des DataArrrays bestimmt.
WIrd **keine** Keyfunction angegeben sind die Keys die Indices des Arrays.
Die keys werden dann verglichen mit den Keys der **selectAll**-function. Alle data-keys die *unterschiedlich* sind zu den keys der selectAll Function werden dann Teil der  **enter()**-function.

**enter():** wenn ein data-element gefunden wurde, das kein UI hat
